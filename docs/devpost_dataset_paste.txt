EVIDENCE DATASET DOCUMENTATION

SIFTGuard was evaluated against a simulated APT intrusion scenario: lateral movement + persistence on Windows Server 2019 (hostname: CORP-WIN-SRV01, incident date: 2024-01-15).

SUPPORTED EVIDENCE FORMATS
- Windows Memory Dump (.mem, .raw, .dmp) — analyzed via Volatility3
- Windows Event Logs (.evtx) — analyzed via python-evtx
- Disk Images (.E01, .dd, .img) — analyzed via Sleuthkit (fls, mmls)
- Network Captures (.pcap) — planned, not yet implemented

SIMULATED DATASET (DEMO_MODE=true)
When real evidence files are absent, SIFTGuard activates forensic simulation mode — generating realistic DFIR artifacts that mirror a real intrusion.

Simulated files:
  data/evidence/memory/victim.mem    (2 GB Windows memory dump)
  data/evidence/logs/Security.evtx  (20 MB, ~11,500 events)
  data/evidence/logs/System.evtx    (10 MB, ~6,000 events)
  data/evidence/disk/victim-disk.E01 (50 GB disk image)
